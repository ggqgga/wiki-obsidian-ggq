# raw/ — 불변 원본 저장소

> 사용자가 수집한 모든 원본 자료가 보관되는 곳. **AI는 이 폴더의 파일을 절대 수정/삭제하지 않는다.**

## 규칙

- **읽기 전용**: raw/ 안의 파일은 생성만 가능, 수정/삭제 금지
- **원본 보존**: 클리핑, 복사, 다운로드한 그대로 저장
- **유형별 분류**: 하위 폴더에 소스 유형별로 정리

## 폴더 구조

| 폴더 | 내용 | 예시 |
|------|------|------|
| `my_folder/` | 사용자가 직접 쓴 글 | 생각 정리, 메모, 아이디어, 일지 |
| `sources/` | 외부에서 가져온 모든 텍스트 | 아티클, 논문, 책, YouTube, 팟캐스트 등 |
| `assets/` | 이미지, PDF, 첨부 파일 | 다이어그램, 스크린샷 |

> 세부 분류(article, book, youtube, paper 등)는 frontmatter의 `tags`로 구분한다. 폴더는 단순하게, 분류는 메타데이터로.

## 파일 네이밍

- kebab-case 사용: `my-article-title.md`
- 날짜 prefix 권장: `2026-04-16-article-title.md`
- 확장자: 텍스트는 `.md`, 나머지는 원본 확장자 유지
