# wiki/ — AI가 컴파일하는 지식 베이스

> raw/의 원본 자료를 AI가 읽고, 요약하고, 연결하여 구축하는 위키. **AI가 소유하고 관리한다.**

## 규칙

- **모든 페이지에 YAML frontmatter 필수**: title, created, updated, tags, source
- **내부 참조는 `[[wikilink]]` 형식만 사용**
- **페이지 생성/삭제 시 index.md 반드시 업데이트**
- **모든 변경 시 log.md에 기록**
- **새 페이지보다 기존 페이지 업데이트 우선** (중복 방지)
- **소스 요약은 사실만 (sources/), 해석은 개념 페이지에서 (concepts/)**

## 폴더 구조

| 폴더 | 역할 | 내용 |
|------|------|------|
| `sources/` | 소스 요약 | raw/ 원본 1개당 요약 페이지 1개 |
| `entities/` | 엔티티 | 인물, 조직, 제품, 프로젝트 |
| `concepts/` | 개념 | 아이디어, 프레임워크, 이론, 분석 |
| `insights/` | 인사이트 | 교차 분석, 비교, 트렌드, 결론 |

## 특수 파일

- **index.md**: 마스터 목차. 모든 페이지가 여기에 등록되어야 함. 항목당 한 줄, 120자 이내.
- **log.md**: 시간순 활동 기록. 모든 ingest/query/maintenance/sync 기록.
