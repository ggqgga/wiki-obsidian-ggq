# Output/ — 최종 결과물

> wiki/의 지식을 기반으로 생성된 최종 산출물. 사업설명서, IR 자료, 기획서, 발표 자료 등.

## 규칙

- wiki/ 페이지를 소스로 활용하여 생성
- 결과물에 사용된 wiki 페이지를 출처로 명시
- 버전 관리: 파일명에 날짜 또는 버전 포함 권장

## 예상 결과물 유형

| 유형 | 형식 | 대상 |
|------|------|------|
| 사업설명서 | Markdown / PDF | 투자자, 파트너 |
| IR 자료 | Markdown / Slide | 투자자 |
| 기획서 | Markdown | 내부 |
| 인포그래픽 | Excalidraw / SVG | 발표, 공유 |
| 다이어그램 | Excalidraw / Mermaid | 내부, 발표 |
| 슬라이드 | Marp / Markdown | 발표 |
